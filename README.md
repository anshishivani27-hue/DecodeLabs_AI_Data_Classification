# 🤖 AI Data Classification

## 📌 Description

This project is a beginner-friendly Machine Learning application developed using Python and Scikit-learn. It trains a Decision Tree Classifier using the Iris dataset and predicts the species of a flower based on sample data.

## ✨ Features

- Loads the Iris dataset
- Splits data into training and testing sets
- Trains a Decision Tree Classification model
- Calculates model accuracy
- Allows users to choose sample data for prediction
- Displays the predicted flower species

## 🛠 Technologies Used

- Python 3
- Scikit-learn
- Decision Tree Classifier

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python classification.py
```
